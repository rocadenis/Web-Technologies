Învăță SQL

SQL (Structured Query Language) este un limbaj de programare folosit pentru gestionarea bazelor de date relaționale.

Concepte de bază în SQL

  • SELECT: Utilizat pentru a selecta date dintr-o bază de date.
  • INSERT INTO: Utilizat pentru a insera date noi într-o bază de date.
  • UPDATE: Utilizat pentru a actualiza date existente într-o bază de date.
  • DELETE FROM: Utilizat pentru a șterge date dintr-o bază de date.
  • WHERE: Utilizat pentru a filtra datele bazate pe anumite condiții.
  • JOIN: Utilizat pentru a combina datele din mai multe tabele în baza de date.
  • ORDER BY: Utilizat pentru a sorta datele în ordine crescătoare sau descrescătoare.
  • GROUP BY: Utilizat pentru a grupa datele bazate pe valori comune într-o coloană.
