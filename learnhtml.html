Începe să înveți HTML

HTML (HyperText Markup Language) este limbajul standard folosit pentru a crea și a structura paginile web.

Elemente de bază HTML

  • <!DOCTYPE html> - Definirea tipului de document HTML.
  • <html> - Elementul rădăcină al documentului HTML.
  • <head> - Conține informații despre documentul HTML.
  • <meta> - Furnizează metadate despre documentul HTML.
  • <title> - Definește titlul documentului HTML.
  • <body> - Conține conținutul vizibil al documentului HTML.
  • <h1>, <p>, <ul>, <li> - Elemente pentru structurarea și organizarea conținutului.
